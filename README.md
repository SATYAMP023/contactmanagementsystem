# Contact Management System

A desktop application built in Java for managing contacts efficiently with persistent data storage using JSON files.

## Features

- User registration and login system
- Add, edit, and delete contact entries
- Store contact information including name, phone number, email, and address
- Search and filter contacts
- Data persistence using JSON file integration
- User-friendly GUI built with Java Swing
- Secure password handling

## Tech Stack

- **Language:** Java (100%)
- **GUI Framework:** Java Swing (javax.swing)
- **Data Storage:** JSON files
- **IDE:** NetBeans

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- NetBeans IDE (recommended)

### Installation

1. Clone the repository:
    ```
    git clone https://github.com/SATYAMP023/contactmanagementsystem.git
    ```
2. Open the project in NetBeans IDE
3. Build the project using the build.xml configuration
4. Run the application from the main class

## Usage

1. Launch the application
2. Register a new user account or login with existing credentials
3. Add contacts with their details (name, phone, email, address)
4. Search for specific contacts using the search functionality
5. Edit or delete contacts as needed
6. All data is automatically saved to JSON files

## Features in Detail

- **Login System:** Secure authentication with username and password
- **Contact CRUD Operations:** Create, Read, Update, and Delete contact information
- **JSON Integration:** Lightweight and efficient data storage
- **Graphical User Interface:** Intuitive Swing-based interface with custom styling
